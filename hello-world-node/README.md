# Hello World App

This project is the Node.js version of the Hello World app.
Although it is a basic example, it is fun to see the Hello World version of each programming language (or runtime, in this case)! :smile:

## Available Scripts

In the project directory, you can run:

### `npm install`

Installs the app dependencies, if any.

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm test`

Runs the tests.
